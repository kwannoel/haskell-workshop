# Haskell workshop

# Requirements

- Install haskell toolchain: 

  https://www.haskell.org/platform/

  We will need `ghci`.

- Clone the project:
  
    ```sh
    git clone git@github.com:kwannoel/haskell-workshop.git
    ``` 

# Running + Compiling the files

``` sh
ghci <file>
```
